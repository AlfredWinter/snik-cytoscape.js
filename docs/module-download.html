(static) downloadPng(full, highRes)

Source:
  • download.js, line 75
Download the graph as a PNG (lossless compression).
Parameters:
Name Type Description
full boolean Iff true, include the whole graph, otherwise only include what is inside the canvas boundaries.
highRes boolean Iff true, generate a high resolution picture using the maximum width and height from config.js. Otherwise, either use the native resolution of the canvas (full=false) or the standard resolution (full=true) from config.js.
